InsClaw（品牌标记 yingshi）— 桌面图标（由品牌图生成）

当前默认渠道为 yingshi / InsClaw：安装脚本会根据 static/branding/brands.json 中 marks.yingshi.install 使用
  static/yingshi_box.ico（Windows）、InsClaw.lnk 名称等。
若仍为必火渠道（marks.bihu），对应源图为 static/bihu_source.png（白底「必火AI BOX」龙虾形象），生成 bihu_box.ico、bihu_*.png；替换请先覆盖源图再执行 regen_icons.sh。

生成文件（必火素材路径，兼容旧品牌）：
  yingshi_box.ico / yingshi_*.png — InsClaw 网页与安装引用（参见 brands.json）
  bihu_box.ico       — Windows 快捷方式 / 文件夹 / 应用图标（bihuo）
  bihu_16.png … bihu_1024.png — 网页 favicon / bihuo 文档引用

一键重生必火尺寸资源（macOS，需 Xcode 命令行工具：sips、iconutil）：
  cd static && chmod +x regen_icons.sh && ./regen_icons.sh

Windows 安装：install.bat / install_slim.bat 结束时会按 LOBSTER_BRAND_MARK 创建桌面快捷方式（默认 InsClaw）。跳过：安装前设置环境变量 LOBSTER_SKIP_DESKTOP_SHORTCUT=1。

Windows 手动换图标：右键快捷方式 → 属性 → 更改图标 → 浏览 → 选对应品牌的 .ico（yingshi 为 static\yingshi_box.ico）。
