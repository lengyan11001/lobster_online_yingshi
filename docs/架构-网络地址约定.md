## upstream_urls.json 与在线版

仓库中的 `upstream_urls.json` 可能含单机/调试用的直连 `api.xskill.ai`。**当 `LOBSTER_EDITION=online` 且已配置 `AUTH_SERVER_BASE` 时**，本机 MCP 在 `mcp/http_server.py` 中**强制**将 `sutui` 上游设为 `{AUTH_SERVER_BASE}/mcp-gateway`，不采用文件中的直连地址，与 `pack_bundle.env` 一致。

经 `/mcp-gateway` 转发时，目标为**服务器上的 lobster MCP**（与 `xskill` 裸 MCP 不同），**仅支持 `invoke_capability` 工具名**。本机在线版在走 gateway 时会自动改为调用 `invoke_capability`（携带 `capability_id`/`payload`），并解析 JSON-RPC 返回，不再向上游发送裸 `generate` 等工具名。
